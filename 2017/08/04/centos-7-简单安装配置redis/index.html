centOS 7 简单安装配置Redis

2017-08-04 · 50 words · 1 minute read
tools
database · Linux

 centOS7上Redis的安装与简单配置.

Note:This blog without all the resources that are outside the original,reproduced please indicate the source.

换系统了重新配置,redis最新版本下载地址在这里，至于稳定版本还是最新的可选择下载，推荐stable。

下载好后: 为redis创建一个文件夹：

mkdir ～/Redis/

进入下载目录将下载好的压缩包放到刚刚创建的文件夹里

sudo mv redis-4.0.1.tar.gz ~/Redis/

进入Redis目录cd ~/Redis/ 解压文件：

sudo tar -zxf redis-4.0.1.tar.gz

进入刚刚解压缩的目录：cd redis-4.0.1/

执行命令：make 如无意外安装成功，进入src目录下，运行命令：

./redis-server

即可开启redis服务。

让Redis后台启动，还需修改以下配置文件：

上面的操作中我们在src目录下，此时我们返回上级目录（即解压包），能看到有一个redis.conf的文件，对这个文件进行修改：

vim redis.conf

在文件中找到daemonize，把no改为yes。保存退出。

再进入src文件夹，开启redis服务，并且加载刚刚我们修改了的配置文件，执行如下命令：

./redis-server ~/Redis/redis-4.0.1/redis.conf

此时完成后台启动，进入client即可开始使用： 运行：./redis-cli

建议：每个人的使用习惯都不一样，安装部署思路是这样的，安装位置读者可按自己喜好设置。
