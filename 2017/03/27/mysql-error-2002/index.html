MySql ERROR 2002

2017-03-27 · 79 words · 1 minute read
tools
database

 MySql-> ERROR 2002 (HY000): Can’t connect to local MySQL server through socket ‘/tmp/mysql.sock’ (2)

Note:This blog without all the resources that are outside the original,reproduced please indicate the source.

今天突然就遇到了Mysql登录不上,报错

ERROR 2002 (HY000): Can't connect to local MySQL server through socket '/tmp/mysql.sock' (2)

网上搜这个错误有很多很多,我说说我遇到的情况.

  1. 终端连接不上,报错ERROR 2002 (HY000): Can’t connect to local MySQL server through socket ‘/tmp/mysql.sock’ (2)
  2. 打开mysql workbench,发现所有库都没有连接
  3. (我用的mac)在设置里看了一下,mysql server没有启动,启动mysql server失败!
  4. 接3,仔细一看,有个警告Warnning:the /usr/local/mysql/data directory is not owned by the mysql如图
  5. 问题就出在这,我不久前改了/usr/local宿主….

解决: $ sudo chown -R mysql /usr/local/mysql/data
